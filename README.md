# squarify

アップロードされた画像に背景をつけて正方形の画像で返すWebアプリケーションです。

## 特徴

ボタン押すだけラクラク

## 注意

雑実装。同時に複数人が使うと他人の上げた画像が返ってくる可能性がある。

## How to use

1. `yarn` (or npm if you like)
1. `pm2 start app.js` (or another if you like)
1. Visit http://[address]:1234
